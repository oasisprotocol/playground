## Running Locally

To run the Oasis Playground webapp on your local machine:

1. Install dependencies:
   ```bash
   yarn
   ```

2. Start the development server (automatically generates projects.json):
   ```bash
   yarn dev
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:5173/
   ```
